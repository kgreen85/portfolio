Conference Go

This page contains the case study of Conference Go which includes the Project Overview, Tools Used and a GitLab Link to the code.

Project Overview

This is an app for creating and attending conferences. It features the ability to create a new conference and its location, sign up presenters with the ability for the admin to approve or deny the presentation with automated email response and sign up to attend a conference.

The app is built on Django with a RESTful API backend. It also uses two 3rd party api's run trough an anti-corruption layer. The api's are used to dynamically generate the conference picture and current weather data based on location. The front-end is build using JavaScript and React.
